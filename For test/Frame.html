Frame tag, <frameset> & <frame>

        <frameset cols="20%, 80%">
            <frame SRC="ex6-1-2.html" name="first" />
            <frame SRC="ex6-1-6.html" name="second" />
        </frameset>

Not supported in HTML5

Only use the frame tag in file

        <a href="test.html" target="first">Test</a>

If you click that <a> tag, than the file is open at the <frame> that name is "first"

<frameset> cols rows border frameborder framespacing
<frame> src name frameborder marginwidth marginheight scrolling
