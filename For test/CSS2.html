Display

    Block
    • Full-line
    • Line break is added before/after it
    • Can set the size.
    Inline
    • Take up as much as width as necessary
    • Does not line break
    • Cannot set the size.

:first-chlid & :nth-child pseudo-class

        p:first-child{          /*any <p> element that is the first child of any element*/
            color: blue;
        }
        p i:first-child{        /*the first <i> element in all<p> elements*/
            color: red;
        }
        p>i{                    /*any <i> element in all<p> elements*/
            color: green;
        }
        p:nth-child(2){         /*any <p> element that is the 2nd child of any element*/
            color: yellowgreen;
        }
        p:nth-child(2) i{       /*all <i> elements in <p> elements that are the 2nd child of any element*/
            color: yellow;
        }
        p:nth-child(2) i:nth-child(2){       /*any <i> elements that is the 2nd child of <p> elements that are the 2nd child of any element*/
            color: orange;
        }
